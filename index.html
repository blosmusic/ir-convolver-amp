Audio Input to Guitar Cab IRs

Click to Enable Audio

The inputs allow users to select the input source and Impulse Response of a select number of guitar cab speakers.

Other FX, routing, and signal chains can be added to the audio signal to create a more complex audio chain.

Impulse Responses are used to simulate the sound of a guitar cabinet and are from the Allure IR Pack.

NOTE: Headphones is suggested if using a devices speaker as both input and output to stop feedback.
